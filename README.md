# 第4次スーパーロボット大戦 改造エディタ (srw4edit PWA)

SFC「第4次スーパーロボット大戦」(V1.1) のROM改造エディタ。ブラウザだけで動きます。

**→ https://kohtatthin.github.io/srw4edit/**

## 特徴

- パイロット / ユニット（敵含む全機） / 武器 / 強化パーツ / 改造テーブル / 主人公誕生日 を編集
- **ROMファイルはどこにも送信されません**。処理はすべてブラウザ内（Pyodide + Python wheel）で完結
- iPhone対応: Safariで開いて「ホーム画面に追加」するとアプリとして使えます
- オフライン対応: 一度開けば以後は圏外でも起動します（service worker）

## 使い方

1. ページを開く（初回はPyodideのダウンロードで少し待つ）
2. 手持ちの ROM ファイル（ヘッダなし 3,145,728 bytes、ヘッダ付きも自動処理）を選択
3. 一覧から選んで編集 → 保存
4. 「改造ROMをダウンロード」で .smc を書き出し

## 構成

- `index.html` — エディタ本体（UI + Pyodide 統合）
- `srw4edit-*.whl` — ROM解析・書き戻しロジック（Python）
- `sw.js` — オフラインキャッシュ

ROM本体・ゲームデータは含みません。
